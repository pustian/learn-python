### 条件

##### if-else

```python 
if condition_1:
    statement_block_1
else:
    statement_block_2
```

##### if-elif

```python
if condition_1:
    statement_block_1
elif condition_2:
    statement_block_2
else:
    statement_block_3
```

### 循环

##### while

```python
while 判断条件(condition)：
    执行语句(statements)……
```

##### while 循环使用 else 语句

```python
while <expr>:
    <statement(s)>
else:
    <additional_statement(s)>
```

##### for 语句

```python
for <variable> in <sequence>:
    <statements>
else:
    <statements>
```

##### 循环嵌套

```python
for iterating_var in sequence:
   for iterating_var in sequence:
      statements(s)
   statements(s)

while expression:
   while expression:
      statement(s)
   statement(s)
```
